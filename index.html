Wallpaper Explorer

Dieses Projekt verwendet eine öffentliche API, um Hintergrundbilder anzuzeigen. Benutzer können durch verschiedene Bilder stöbern und neue Wallpapers laden.

Technische Spezifiken Information

Erklaerung der Website

Wenn man auf der Navbar auf Wallpaper drueckt, dann werden eben random wallpaper geloaded, jedoch kann man dann noch bisschen filtern und suchen nach bestimmten sachen. Der API ohne Verifizierung kann nicht alle Bilder loaden, jedoch genug um einen schoenen Hintergrund zu finden.
